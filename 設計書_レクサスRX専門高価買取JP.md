## 4. デザインシステム（★ゴールド×ブラック 高級路線）

ランクル300より**高級感を1ランク上げる**。マットブラック地に上質なシャンパンゴールドの差し色、細い金ラインの装飾、余白広め、セリフ見出しで品格を演出。**モバイルファースト必須**。

### 4.1 カラー（CSS変数で定義）
```css
:root{
  --bg:           #0a0a0a;   /* ベース漆黒 */
  --bg-soft:      #141414;   /* セクション地 */
  --surface:      #1c1a16;   /* カード（黒に微かな茶味） */
  --gold:         #c8a253;   /* メインゴールド（シャンパン） */
  --gold-bright:  #e7c87a;   /* ハイライト金 */
  --gold-deep:    #9c7c33;   /* 陰影金 */
  --line-gold:    rgba(200,162,83,.35); /* 金の細線 */
  --text:         #f4f1ea;   /* 本文（オフホワイト） */
  --text-muted:   #b8b2a6;   /* 補足 */
  --line-cta:     #06c755;   /* LINE緑（CTAのみ・ブランド準拠） */
}
```
- 基調は黒、アクセントは金のみ。多色使い禁止。
- グラデは金の微グラデ（`--gold-deep`→`--gold-bright`）をボタン・見出し下線・バッジ枠に限定使用。

### 4.2 タイポグラフィ
| 用途 | フォント | 指定 |
|---|---|---|
| 英語見出し（セクションラベル） | Cinzel / Cormorant Garamond（セリフ） | letter-spacing広め・金色 |
| 日本語見出し | Noto Serif JP（明朝）で品格 | 太字・白 |
| 本文 | Noto Sans JP | 16px / line-height 1.9 |
| 数字・価格 | Cinzel or 既存sans | 強調は金 |

> ランクル300はゴシック主体。RXは見出しに明朝/セリフを採用し高級感を差別化。

### 4.3 コンポーネント / 装飾
- **ボタン**: 金グラデ背景・黒文字、またはアウトライン金（ホバーで反転）。角丸6px、影は控えめ。
- **カード**: `--surface`地に1px金枠（`--line-gold`）、ホバーで金枠が明るく。
- **見出し装飾**: セクション英語ラベルの両脇に短い金の横線。番号（01〜）は大きめの金アウトライン数字。
- **バッジ**: 帝国データバンク61点は金枠の円形/盾形で強調。
- **区切り**: セクション間に細い金ライン or 金ドット。
- **画像**: RX車両は黒背景に映える明るめの写真、金の細枠 + 控えめシャドウ。

### 4.4 レイアウト
- モバイルファースト。最大コンテンツ幅 1080px、左右余白広め。
- 縦リズムをゆったり（セクション上下 padding 大）で高級感。
- 横スクロール厳禁。タップ領域44px以上。

## 5. レクサスRX 買取対応グレード（要確認・暫定）

現行レクサスRX（5代目 2022年〜）想定。**確定前にクライアント確認推奨**。

- RX500h F SPORT Performance（買取強化候補）
- RX450h+（PHEV）
- RX350h（HEV）
- RX350 F SPORT
- RX350

> ランクル300は「GR SPORT / ZX / VX / AX / GX」。RXは上記HEV/PHEV/F SPORT構成に置換。査定フォームのグレード選択肢・年式（RXは2022〜現行が主、旧型対応可否は要確認）も合わせて差し替える。

### 7.3 デプロイ（★ランクル300の重要修正点を踏襲）
**本番はさくらインターネット**（ランクル300は `lancru300kaitori.jp` がさくら本番）。
- Cloudflare PagesのサンプルURLは**本番ではない**点に注意（ランクル300で発生した認識齟齬の再発防止）。
- 本番ドメイン: 未定（例: `lexusrx-kaitori.jp` 等）→ **要確認**。
- デプロイ運用: ①GitHub push → ②さくらサーバへ `index.html` / `style.css` を上書きアップロード（FTP）。
- `.env.deploy` にさくらFTP情報（host/user/pass/dir）を登録できれば自動化。**未登録なら手動アップロード**。
- アップロード後は必ずスーパーリロード（Cmd+Shift+R / Ctrl+F5、スマホはキャッシュ削除orシークレット）で確認。
